# 크롬 확장 프로그램 설치 방법

## 1. 백엔드 서버 시작

```bash
cd c:/web-agent
python backend/app.py
```

서버가 `http://localhost:5000`에서 실행됩니다.

## 2. 크롬 확장 프로그램 로드

1. Chrome 브라우저에서 `chrome://extensions/` 열기
2. 우측 상단 "개발자 모드" 토글 켜기
3. 좌측 상단 "압축해제된 확장 프로그램 로드" 클릭
4. `c:/web-agent/extension` 폴더 선택
5. 확장 프로그램이 로드됨

## 3. 사용 방법

### 포트폴리오 설정
1. 브라우저 우측 확장 프로그램 아이콘 클릭
2. PDF 파일 선택 후 "PDF 업로드" 클릭
3. 파싱된 정보 확인

### 자기소개서 작성
1. 채용 공고 텍스트 입력
2. 질문 입력 (선택사항)
3. "자기소개서 작성" 클릭
4. AI가 작성한 자기소개서 확인

### 합격률 팁
1. 채용 공고 입력
2. "팁 생성" 클릭
3. 강점/개선점 확인

### 자동 채우기
1. 지원서 페이지로 이동
2. 확장 프로그램 열기
3. "기본 정보 자동 채우기" 클릭
4. 이름, 이메일, 전화번호 자동 입력

## 4. 주의사항

- 백엔드 서버가 항상 실행 중이어야 함
- 포트폴리오는 세션 기반이라 서버 재시작 시 삭제됨
- 작성 완료 후 "포트폴리오 삭제" 버튼으로 데이터 삭제 권장

## 5. 문제 해결

### CORS 에러
- 백엔드 서버가 실행 중인지 확인
- `http://localhost:5000` 접근 가능한지 확인

### 자동 채우기 작동 안함
- 포트폴리오가 업로드되었는지 확인
- 페이지가 완전히 로드된 후 시도
